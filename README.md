# 👸 공주 옷 갈아입히기 게임 (Princess Dress-up)

우리 딸을 위한 특별한 선물! 예쁜 옷과 왕관, 구두를 골라 공주님을 꾸며주세요.

## 시작하는 방법
1. 이 폴더 안에 있는 `index.html` 파일을 크롬이나 사파리 같은 브라우저로 열어주세요.
2. 왼쪽의 옷장 아이템을 클릭하면 캐릭터가 옷을 입습니다.
3. '사진 찍기' 버튼을 눌러 예쁜 공주님의 모습을 칭찬해 주세요!

## 기술 스택
- HTML5, CSS3, JavaScript (Vanilla)
- Google Fonts (Gaegu)
- AI Generated Assets (DALL-E)

## 폴더 구조
- `index.html`: 게임 화면 구조
- `style.css`: 디자인 및 애니메이션
- `script.js`: 게임 로직 (옷 갈아입기, 반짝이 효과)
- `assets/`: 캐릭터 및 아이템 이미지 파일들
